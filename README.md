# E-Commerce Website

This is a fully functional eCommerce website built using Vite and vanilla JavaScript. The project showcases a modern, responsive online store with a focus on performance and user experience.

## Key Features:
- **Fast & Optimized**: Built with Vite, ensuring rapid development and optimized production builds.
- **Dynamic Product Listings**: Display products dynamically with real-time filtering and sorting options.
- **Shopping Cart Functionality**: Add, remove, and update items in the cart with seamless user interactions.
- **Responsive Design**: Mobile-first approach, ensuring the site looks great on all devices.
- **Custom Vanilla JS Components**: No frameworks usedâ€”everything is built from scratch with clean, modular JavaScript.
- **Secure Checkout Process**: Mock payment gateway integration for a complete eCommerce experience.

## Getting Started:
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Use `npm run dev` to start the development server.
4. Build for production with `npm run build`.
